# kyc-risk-analysis-project

Analysis

## Overview
This project simulates a real-world KYC (Know Your Customer) and AML screening process using SQL and Excel.

## Objectives
- Identify high-risk customers
- Detect PEP and sanctions exposure
- Analyze Ultimate Beneficial Owners (UBOs)
- Classify customers into risk levels

## Tools Used
- SQL (SQLite)
- Microsoft Excel

## Key Analysis
- High-risk customer identification using SQL queries
- UBO risk exposure analysis using JOIN operations
- Risk classification using rule-based logic
- Dashboard visualization in Excel

## Key Insights
- Identified customers linked to PEP and sanctions exposure
- Demonstrated how UBO structures impact risk
- Built a simple risk classification model
